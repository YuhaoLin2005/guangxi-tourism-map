# guangxi-tourism-map
基于 ArcGIS Maps SDK for JavaScript 开发的广西全域旅游地图应用，严格满足“页眉+地图区域+底图库区域”的三区域布局要求，完整集成比例尺、图例、搜索框、图层控制等核心微件，同时实现景点评分分级渲染、游客量可视化等拓展功能。

## AI（DeepSeek）辅助开发说明
本项目开发过程中深度借助DeepSeek大模型完成核心开发工作，具体落地场景如下：
1. **界面布局与样式开发**：基于DeepSeek生成的弹性布局CSS代码，完成页眉、地图区域、底图库区域的三区域适配，优化底图库悬停交互、响应式适配、微件自定义样式（比例尺/图例/搜索框美化）等关键样式逻辑；
2. **ArcGIS JS API 逻辑梳理**：通过DeepSeek辅助解析ArcGIS官方文档，完成图层Token授权、多图层搜索微件适配、原生弹窗替换为右侧信息面板等核心逻辑编写，解决图层加载、交互触发等技术问题；
3. **数据可视化功能实现**：依托DeepSeek提供的分级渲染、动态柱状图开发思路，实现景点评分的ClassBreaksRenderer分级渲染、游客量数据的动态柱状图可视化，让专题数据更直观；
4. **问题排查与优化**：借助DeepSeek定位并修复底图库布局空白、搜索结果弹窗冲突、地图加载延迟等问题，提升页面稳定性和交互体验。

## 核心功能
- 三区域页面布局：页眉（图层控制）+ 地图主区域（含比例尺/图例/搜索框）+ 底图库区域；
- 景点数据可视化：基于评分的分级渲染、游客量动态柱状图；
- 交互优化：自定义信息面板取代原生弹窗、底图悬停效果、图层切换反馈；
- 响应式适配：兼容移动端/桌面端显示。
